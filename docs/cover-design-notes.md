# 小红书爆款封面设计要点

## 主要封面类型

### 1. 大字报型（最常见）
- **特点**：简洁背景 + 超大粗体文字
- **背景**：纯色/笔记本纹理/简单图案
- **文字**：黑色粗体为主，关键词用黄色/粉色高亮底色
- **装饰**：小圆点、下划线、高亮色块
- **示例**："后悔没有早点..." + 黄色高亮

### 2. 手绘边框型
- **特点**：手绘风格边框 + 大字文案
- **背景**：白色/浅色
- **边框**：蓝色/彩色手绘线条，带圆环装饰
- **文字**：多行排列，关键词用彩色高亮
- **示例**：蓝色手绘边框 + 黄色/粉色关键词高亮

### 3. 截图对话型
- **特点**：模拟聊天截图/邮件截图
- **背景**：手机界面/聊天气泡
- **文字**：对话形式，真实感强
- **适用**：求助类、经历分享类

### 4. 对比型
- **特点**：before/after 或 左右对比
- **布局**：分栏设计
- **文字**：对比鲜明的标题
- **适用**：成功案例、经验分享

### 5. 速报/新闻型
- **特点**：模拟新闻标题
- **装饰**：爆炸图标、感叹号
- **文字**：紧急感、时效性
- **适用**：警示类、热点类

## 配色方案

### 共鸣型（温暖）
- 主色：#FFF8E1（暖白）
- 高亮：#FFD54F（黄色）
- 文字：#212121（黑色）

### 警示型（醒目）
- 主色：#FFEBEE（浅红）
- 高亮：#FF5252（红色）
- 文字：#212121（黑色）

### 求助型（清新）
- 主色：#E3F2FD（浅蓝）
- 高亮：#2196F3（蓝色）
- 文字：#212121（黑色）

### 成功型（活力）
- 主色：#E8F5E9（浅绿）
- 高亮：#4CAF50（绿色）
- 文字：#212121（黑色）

### 吐槽型（个性）
- 主色：#F3E5F5（浅紫）
- 高亮：#9C27B0（紫色）
- 文字：#212121（黑色）

## 设计原则

1. **文字要大**：主文案字号要足够大，在小图预览时也能看清
2. **高亮关键词**：用色块底色突出核心词汇
3. **简洁背景**：避免复杂背景干扰文字
4. **情绪明确**：配色和文案要传达明确的情绪
5. **留白适当**：不要堆砌太多元素
